# Python/kivy code example
### To start demonstration:

* need python 3.6
* create virtual environment    
 `python3 -m venv`
* activate venv  
`source venv/bin/activate`  
* install requirements   
`pip install -r requirements.txt`  
* to start pong game demo  
`python pong.py`
* to start painter demo  
`python paint.py`
